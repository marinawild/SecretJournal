# Secret Journal
The Secret Journal is a product that protects a user's physical journal through a multi-phase verification, locking system. The journal is locked inside a 3D printed case where it can only be unlocked through the correpsonding web application. The web aplication features a multi-phase athentication system comprising of face recognition and voice recognition. 
## How to Use
When the user wants to access their journal, they press the "Start Verification" button. The application prompts to user to look into the camera and speak the phrase, "Unlock my Secret Journal". The application checks that the user matches the stored face and voice information, and if correct, the box storing the journal is unlocked. The user is taken to a new web page where the application displays a custom journal prompt, using the user's personal information for customization.
